# 关于产品名称——从 SkillMix 改为 AgentMix

> 原决策 7（DESIGN.md v0.6 设计决策附录）

前期候选名包括 SkillMix（最初定名，强调"组合 Skills"）、SkillWeave、SkillForge。当 Phase 4+ 决定扩展到 slash commands、subagents、CLAUDE.md 片段、MCP 配置等更广义的 Agent 资产后，"Skill"前缀名不副实——会让用户误以为工具只处理 SKILL.md。最终改为 AgentMix，传达"任意 Agent 资产的跨项目组合"。名字与 DESIGN.md §3.2 的 `Asset` 抽象、ROADMAP.md Phase 4 的扩展路线呼应。
